The game includes:

- Playing multiple rounds against the computer
- Random computer choices using `rand()`

## 🕹️ How It Works

1. The player chooses:
   - `1` → Stone
   - `2` → Paper
   - `3` → Scissor

2. The computer randomly selects its move.

3. The program compares both choices and determines the winner of the round.

4. The game keeps track of:
   - Player wins
   - Computer wins
   - Draws

5. After all rounds are completed, the final results are displayed along with the overall winner.
